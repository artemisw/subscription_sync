OAuth 設定說明

需要一個同時開啟 Google Sheets API 和 Google Calendar API 的 OAuth Client ID。

STEPS

  1. Cloud Console → 新專案 → 啟用 Sheets API 與 Calendar API
  2. 憑證 → OAuth 2.0 用戶端 ID → Web application → Authorized origins 加入
  3. 取得 Client ID 填入下方，再填 Sheet ID 與工作表名稱
  4. 回到「輸入」頁面，分別登入 Sheets 和 Calendar
